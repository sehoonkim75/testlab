이용 안내 및 유의사항

자기소개서 기반 면접질문 생성기를 사용하기 전에 아래 내용을 확인해주세요.

서비스 개요

지원자의 자기소개서와 모집분야 직무수행내용을 OpenAI API로 분석하여, 평가요소별 맞춤형 면접질문(주질문·근거구절·질문이유·보조질문)을 생성하는 도구입니다. 별도 서버 없이 브라우저에서만 동작합니다.

이용 순서

  1. OpenAI API 키를 입력하고 등록 버튼으로 확정합니다.
  2. 필요 시 시스템 프롬프트를 편집한 뒤 프롬프트 적용으로 확정합니다.
  3. 단건 테스트 탭에서 자소서를 직접 붙여넣어 프롬프트를 튜닝합니다.
  4. 튜닝이 끝나면 엑셀 일괄 처리 탭에서 대량 생성 후 결과 엑셀을 다운로드합니다.

유의사항

  • API 키 보안 — 키는 이 브라우저 탭에만 유지되며 탭을 닫으면 삭제됩니다. 공용 PC에서는 사용 후 반드시 탭을 닫으세요.
  • 비용 발생 — 생성 1건마다 OpenAI API 사용 요금이 부과됩니다. 대량 처리 전 단건 테스트로 결과 품질과 비용을 먼저 확인하세요.
  • 개인정보 취급 — 자기소개서 내용이 OpenAI 서버로 전송됩니다. 기관의 개인정보 처리 방침과 외부 API 이용 규정을 준수하는 범위에서 사용하세요.
  • AI 생성물 검토 — 생성된 질문은 초안입니다. 실제 면접 활용 전 반드시 담당자가 내용의 적절성을 검토·수정해야 합니다.
  • 처리 중 유지 — 일괄 처리 중 탭을 닫거나 새로고침하면 진행 중인 결과가 사라집니다. 완료 후 결과 엑셀을 즉시 저장하세요.
